## 🎨 Features

- **Bold, Premium Design** — Navy & Gold color palette with glass-morphism cards
- **Full Responsive** — Optimized for desktop, tablet, and mobile devices
- **Social Media Ready** — Open Graph and Twitter Card meta tags for rich link previews
- **Interactive Elements** — Typing effect, scroll progress bar, mouse tilt on hero image
- **Auto-Moving Carousels** — Testimonials carousel with smooth transitions
- **Tool Stack Showcase** — 25+ tools displayed with clickable anchor cards
- **Impact Metrics** — Animated stat counters for Happy Customers, Award Wins, and Complete Projects
- **Booking Integration** — Calendly, WhatsApp, Email, and CV download options
- **SEO Optimized** — Meta descriptions, keywords, and structured data
- **Smooth Animations** — Fade-in scroll effects and hover transitions

🧑‍💻 Developer Notes
The portfolio uses CSS custom properties for easy theming

All colors are defined in the :root section of style.css

Glass-morphism cards use backdrop-filter: blur(12px)

The typing effect cycles through 3 phrases

Auto-sliding carousels transition every 4 seconds

Stat counters animate when scrolled into view
